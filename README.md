# TLT

#### TLT is a simple WebSite created with Django for back-end and Materializecss framework for front-end

# Run project
 ```sh
$  cd blog
```

 ```sh
$  python manage.py makemigrations
```

 ```sh
$  python manage.py migrate
```

 ```sh
$  python manage.py runserver
```
